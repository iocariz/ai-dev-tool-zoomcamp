# Snake Duel Backend

## Prerequisites

- Python 3.12+
- `uv` package manager

## Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```

2. Install dependencies:
   ```bash
   uv sync
   ```

## Running the Server

Start the development server with auto-reload enabled:

```bash
uv run main.py
```

The API will be available at `http://0.0.0.0:3000`.

## Running Tests

Execute the test suite using `pytest`:

```bash
uv run pytest
```
